# Each plugin and why.


| Plugin | Why |
| -------------- | --------------- |
| comment.nvim | easier, faster, smarter comments|
| nvim-surround | fast surrounding manipulation (quotes, brackets, tags)|
| nvim-treesitter | better syntax highlighting and parsing|
| telescope.nvim | fuzzy finder for files, buffers, commands|
| gitsigns.nvim | git status in sign column, hunks, blame|
